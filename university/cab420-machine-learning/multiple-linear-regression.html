Multiple Linear Regression

Definition​

Predicting one thing from several things

Still fitting a line to some data, just in multiple dimensions.

The line:

y=β0+β1x1+β2x2+⋯+βpxpy = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dotsb + \beta_p x_py=β0​+β1​x1​+β2​x2​+⋯+βp​xp​

or

y=β0+∑i=1pβixiy = \beta_0 + \sum^p_{i=1} \beta_i x_iy=β0​+i=1∑p​βi​xi​
  • Many predictors/inputs (x1,x2,x3,…x_1, x_2, x_3, \dotsx1​,x2​,x3​,…, x_p)
  • One response/output (yyy)
  • Need to find β0,β1,β2,…,βp\beta_0, \beta_1, \beta_2, \dots, \beta_pβ0​,β1​,β2​,…,βp​
info

We aim to find values for β\betaβ that minimizes

∑i=1M(yi−y^)2=∑i=1M(yi−(β0+∑j=1Pβj×xij))2\sum^{M}_{i = 1} {(y_i - \hat{y})^2} = \sum^{M}_{i = 1} {\left( y_i - (\beta_0 + \sum^{P}_{j = 1} {\beta_j \times x_{ij}}) \right)^2}i=1∑M​(yi​−y^​)2=i=1∑M​(yi​−(β0​+j=1∑P​βj​×xij​))2
  • yiy_iyi​ is the actual value we are trying to predict (ground truth).
  • yi^\hat{y_i}yi​^​ is the prediction made by the model.
  • MMM is the number of examples.
  • iii stands for the i-th example.
  • jjj stands for the j-th attribute/dimension of an example. (e.g., wind speed, gender)
  • PPP stands for the number of attributes/dimensions of an example.
  • βj\beta_jβj​ is the weight for that attribute/dimension. It is also known as coefficient.
  • xijx_{ij}xij​ stands for the j-th attribute/dimension of the i-th example.

Standard Error​

Definition: A measure of how much the coefficient changes by if we resample the data and recompute the regression.

tStat​

Definition: CoefficientstandardError\frac{Coefficient}{standard Error}standardErrorCoefficient​ indicates that the result is less likely to be the result of noise. Therefore, we want this to be long way from 0.

p-value​

On an individual term, the p-value is the most useful. The null hypothesis states that the coefficient is equal to 0.

  • Low p-value means that you can reject the null hypothesis. Because that term is significant and thus important to the model.
  • High p-value means that the term is not significant, there are two main reasons for this.
    1. the predictor term (input) and the response are not related. In this case, the term shouldn't be in the model
    2. the predictor term is correlated with another predictor. Therefore, the relationship between that variable and the response is captured twice in the model.

Correlation​

When an individual term's p-value is high, using correlation can help identify what's going on to improve the model. see correlation

R2R^2R2 and RMSE are useful values to consider

R-Squared​

Definition: The R2R^2R2 indicates how much of the observed variance explained by the model.

  • 0 - the model captures/explains nothing
  • 1 - the model captures/explains everything
    • 1 is the ideal performance, but 1 may also indicate overfitting.
    • The R2R^2R2 can only be calculated on the training data. but why?

Defined as follows:

  • R2=1−SSESSYR^2 = 1 - \frac{SSE}{SSY}R2=1−SSYSSE​
  • SSE=∑i=0n(yi−yi^)2SSE = \sum^n_{i=0}(y_i - \hat{y_i})^2SSE=∑i=0n​(yi​−yi​^​)2, SSE is the sum of the errors between the predicted and the actual value.
  • SSY=∑i=0n(yi−yˉ)2SSY = \sum^n_{i=0}(y_i - \bar{y})^2SSY=∑i=0n​(yi​−yˉ​)2, SSY or SST (Total Sum of Squares). Is the sum of the differences from the mean.

Adjusted R-Squared​

Definition: Considers the model's complexity (number of terms) as well as how much variance it explains. Which means that complex models are penalized.

RMSE (Root Mean Square Error)​

Definition: Root Mean Square Error (RMSE) is the standard deviation of the residuals (prediction errors). It is a measure of how spread these residuals are. Therefore, smaller RMSEs are better. The scale of RMSE is dependent on the data. Unlike R2R^2R2, it can be computed on training, validation, and testing sets.

RMSE=(prediction−actual)2‾RMSE = \sqrt{ \overline{ (prediction - actual)^2 } }RMSE=(prediction−actual)2​​
def compute_RMSE(predicted, actual):
    return numpy.sqrt(numpy.mean((predicted - actual)**2))

Df Residuals​

Definition: The df(Residual) is the total number of observations (rows) minus the number of parameters being estimated.

F-statistic​

Definition: A p-value used to indicate if the entire model is significant, much the same as we have for the individual terms.
