Cartesian Product​

Given sets AAA and BBB we define the Cartesian Product to be the set

A×B={(a,b):a∈A, b∈B}A \times B = \{(a,b) : a \in A,\ b \in B\}A×B={(a,b):a∈A, b∈B}

The size of A×BA \times BA×B is given by ∣A×B∣=∣A∣×∣B∣|A \times B| = |A| \times |B|∣A×B∣=∣A∣×∣B∣

More Cartesian Products​

More generally we have:

  • A1×⋯×An={(a1,a2,…an):a1∈A1,…an∈An}A_1 \times \dots \times A_n = \{(a_1, a_2, \dots a_n) : a_1 \in A_1, \dots a_n \in A_n \}A1​×⋯×An​={(a1​,a2​,…an​):a1​∈A1​,…an​∈An​}
  • An=A×A×AA^n = A \times A \times AAn=A×A×A (n copies of AAA) - e.g. R2\mathbb{R}^2R2
  • The size ∣A×B×C∣=∣A∣⋅∣B∣⋅∣C∣|A \times B \times C |= |A| \cdot |B| \cdot |C|∣A×B×C∣=∣A∣⋅∣B∣⋅∣C∣

Some Examples​

  • R2\mathbb{R}^2R2 describes points on a 2-dimensional plane.
  • {0,1}n\{0, 1\}^n{0,1}n is the set of bit strings of length nnn.
  • {0,…,1919}×{0,…,1079}\{0, \dots, 1919\} \times \{0,\dots, 1079\}{0,…,1919}×{0,…,1079} encodes (x,y) coordinates on a 1080p screen.

Relations​

Let AAA and BBB be sets. A binary relation from AAA to BBB is a subset of A×BA \times BA×B.

A relation over AAA is a subset of A×AA \times AA×A.

If RRR is a binary relation then we write aRbaRbaRb to mean (a,b)∈R(a,b) \in R(a,b)∈R. So a<ba < ba<b is shorthand for (a,b)∈<(a,b) \in <(a,b)∈<.

Properties of Relations​

Here are some special properties that a binary relation can have:

  • symmetric
  • reflexive
  • transitive
  • anti-symmetric
  • irreflexive

properties of relations

Symmetric​

We say a binary relation is symmetric if

(a,b)∈R  ⟹  (b,a)∈R(a,b) \in R \implies (b,a) \in R(a,b)∈R⟹(b,a)∈R

That is whenever we have (a,b) we also have (b,a)

Examples
  • ===
  • a≡b (mod n)a \equiv b \ (\text{mod}\ n)a≡b (mod n)
  • ∅,A×A\empty, A \times A∅,A×A

Anti-Symmetric​

A relation RRR on a set AAA is anti-symmetric if ∀a,b∈A\forall a,b \in A∀a,b∈A if (a,b)∈R(a,b) \in R(a,b)∈R and (b,a)∈R(b,a) \in R(b,a)∈R, then a=ba=ba=b.

∀a∀b ((a,b)∈R∧(b,a)∈R  ⟹  a=b)\forall a \forall b\ ((a,b) \in R \land (b,a) \in R \implies a=b)∀a∀b ((a,b)∈R∧(b,a)∈R⟹a=b)
Examples
  • R1={(a,b)∈A×A:a=b}R_1 = \{(a,b) \in A \times A : a=b \}R1​={(a,b)∈A×A:a=b} - e.g., a=2, b=2a=2 ,\ b=2a=2, b=2
  • R2={(a,b)∈A×A:a≤b}R_2 = \{(a,b) \in A \times A : a \leq b \}R2​={(a,b)∈A×A:a≤b} - e.g., a=2, b=2a=2, \ b=2a=2, b=2
  • R3={(a,b)∈A×A:a>b}R_3 = \{(a,b) \in A \times A : a > b \}R3​={(a,b)∈A×A:a>b} - cannot find any pair that satisfies the condition. So \implies will always return True.

Not anti-symmetric:

  • R4={(a,b)∈A×A:a+b≤3}R_4 = \{(a,b) \in A \times A : a + b \leq 3 \}R4​={(a,b)∈A×A:a+b≤3}

a=1, b=2a=1, \ b=2a=1, b=2

(1,2)∈R∧(2,1)∈R  ⟹  a=b(1,2) \in R \land (2,1) \in R \implies a = b(1,2)∈R∧(2,1)∈R⟹a=b This returns False, so R4R_4R4​ is not anti-symmetric.

Reflexive​

A binary relation is reflexive iff (a,a)∈R(a,a) \in R(a,a)∈R for every a∈Aa \in Aa∈A

Examples
  • ≤,≥\leq, \geq≤,≥
  • ===
  • ⊆\subseteq⊆
  • a≡b (mod n)a \equiv b \ (\text{mod}\ n)a≡b (mod n)

Irreflexive​

A binary relation R=A×AR = A \times AR=A×A is irreflexive if

∀a∈A (a,a)∉R\forall a \in A \ (a,a) \not \in R∀a∈A (a,a)∈R
Examples

  • ⊂\sub⊂
  • ≠\not ==

Transitive​

A relation RRR on a set AAA is transitive if ∀a,b,c∈A\forall a,b,c \in A∀a,b,c∈A

(a,b)∈R∧(b,c)∈R  ⟹  (a,c)∈R(a,b) \in R \land (b,c) \in R \implies (a,c) \in R(a,b)∈R∧(b,c)∈R⟹(a,c)∈R
Examples
  • R1={(a,b):a=b}R_1 = \{(a,b) : a=b \}R1​={(a,b):a=b}
    • let a=2, b=2, c=2a=2,\ b=2, \ c=2a=2, b=2, c=2

  • R2={(a,b):a≤b}R_2 = \{(a,b) : a \leq b \}R2​={(a,b):a≤b}
    • let a=2, b=5, c=8a=2,\ b=5, \ c=8a=2, b=5, c=8
    • (2,5)(5,8)  ⟹  (2,8)∈R(2,5)(5,8) \implies (2,8) \in R(2,5)(5,8)⟹(2,8)∈R ✅
  • R3={(a,b):a>b}R_3 = \{(a,b) : a > b \}R3​={(a,b):a>b}
    • let a=3, b=2, c=1a=3,\ b=2, \ c=1a=3, b=2, c=1
    • (3,2)(2,1)  ⟹  (3,1)∈R(3,2)(2,1) \implies (3,1) \in R(3,2)(2,1)⟹(3,1)∈R ✅
  • R4={(a,b):a+b≤3}R_4 = \{(a,b) : a + b \leq 3\}R4​={(a,b):a+b≤3}

Equivalence Relation​

An equivalence relation is a binary relation that is:

  • symmetric
  • reflexive
  • transitive
Examples
  • ===
  • a≡b (mod n)a \equiv b \ (\text{mod}\ n)a≡b (mod n)
  • A×AA \times AA×A
Example

Let RRR be the relation on the set of real numbers such that a R ba \ R\ ba R b iff a−ba-ba−b is an integer. Is this an equivalence relation?

R={(a,b):a−b∈Z}R = \{(a,b) : a - b \in \mathbb{Z}\}R={(a,b):a−b∈Z}

  • symmetric? - aRb  ⟹  bRaaRb \implies bRaaRb⟹bRa
    • a−b∈Z  ⟹  b−a∈Za-b \in \mathbb{Z} \implies b-a \in \mathbb{Z}a−b∈Z⟹b−a∈Z
  • reflexive? - aRa  ⟹  a−a∈ZaRa \implies a - a \in \mathbb{Z}aRa⟹a−a∈Z
  • transitive? - aRb∧bRc  ⟹  aRcaRb \land bRc \implies aRcaRb∧bRc⟹aRc
    • a−b∈Z∧b−c∈Z  ⟹  a−c∈Za - b \in \mathbb{Z} \land b - c \in \mathbb{Z} \implies a - c \in \mathbb{Z}a−b∈Z∧b−c∈Z⟹a−c∈Z

Partial Ordering​

A partial ordering on a set AAA is a binary relation over AAA which is:

  • reflexive
  • transitive
  • anti-symmetric
Examples
  • ≤,≥\leq, \geq≤,≥
  • ⊆\subseteq⊆
info

If the relation is irreflexive instead of reflexive, it is called a strict partial ordering. (without the equal sign)

Total Ordering​

A total ordering is a partial ordering RRR over AAA that also has the property:

∀x,y∈A(xRy∨yRx)\forall x,y \in A (xRy \lor yRx)∀x,y∈A(xRy∨yRx)

This means that we can always compare any two elements of AAA.

Example
  • ≤,≥\leq, \geq≤,≥
  • lexicographical ordering
info

If the relation is irreflexive instead of reflexive, it is called a strict total ordering. (without the equal sign)
