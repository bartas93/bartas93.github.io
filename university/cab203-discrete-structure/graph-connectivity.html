Graph Connectivity

Path​

A path is a sequences of vertices [v1,v2,v3,…,vj][v_1, v_2, v_3,\dots, v_j][v1​,v2​,v3​,…,vj​] such that:

  • No vertex appears more than once.
  • vkv_kvk​ is adjacent to vk+1v_{k+1}vk+1​. for each k=1…j−1k = 1 \dots j-1k=1…j−1.
  • The length of the path is j−1j-1j−1 which is the number of edges.
Example

  • A,D,C,BA,D,C,BA,D,C,B is a path of length 3.
  • A,CA,CA,C is a path of length 1.
  • D,C,AD,C,AD,C,A is a path of length 2.

Cycle​

A cycle is just like a path, but v1=vjv_1 = v_jv1​=vj​, so that it loops back on itself.

  • The length of a cycle is j−1j-1j−1, which is the number of edges, and also the number of vertices.
  • We don't care about the starting vertex of a cycle. So A,B,C,AA,B,C,AA,B,C,A is the same cycle as B,C,A,BB,C,A,BB,C,A,B.
  • We might not use the above notation (duplicate the start/end vertex) to represent a cycle. Instead we will use "cycle A,B,CA,B,CA,B,C". Note: this is different from path, path can be represented by "path A,B,CA,B,CA,B,C".
Example

  • A,D,C,AA,D,C,AA,D,C,A is a cycle of length 3.

Connectedness​

If there is a path from uuu to vvv for every u,v∈Vu,v \in Vu,v∈V, then we say that GGG is connected. Otherwise, we say GGG is disconnected.

If for every u,v∈S⊆Vu,v \in S \subseteq Vu,v∈S⊆V there is a path between uuu and vvv, and there are no paths to any vertices outside SSS.

Distance​

The distance between vertices uuu and vvv is the length of a shortest path from uuu to vvv.

def distance_classes(V:set, E:set, u):
    """
    V: a set of vertices.
    E: a set of edges.
    u: the starting vertex.

    return: A list D. The vertices in D[1] is 1 distance from u.
    """
    D = [{u}]
    V_unexplored = V - {u}

    return distance_classesR(V_unexplored, E, D)

def distance_classesR(V,E, D):
    V_unexplored = V - D[-1]
    if len(V_unexplored) == 0:
        return D
    D_new = NS(V_unexplored, E, D[-1])
    D = D + [D_new]
    return distance_classesR(V_unexplored, E, D)

def NS(V,E,S):
    return {v for u in S for v in V if (u,v) in E}

V = {"A", "B", "C", "D", "E"}
E = { 
    ("A","B"),("B","A"),
    ("A","C"),("C","A"),
    ("B","C"),("C","B"),
    ("B","D"),("D","B"),
    ("D","E"),("E","D") 
    }
print(distance_classes(V,E, "A"))
# [{'A'}, {'C', 'B'}, {'D'}, {'E'}]

Bipartite Graph​

If we are able to separate/partition vertices into two sets (A,B)(A,B)(A,B) such that vertices within each set are not adjacent, then we call (A,B)(A,B)(A,B) a bipartition of GGG.

Theorem

A graph GGG is bipartite iff GGG has no cycles of odd length.
