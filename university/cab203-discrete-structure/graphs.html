Graphs

A graph is an irreflexive and symmetric relation.

  • A graph GGG is a pair (V,E)(V,E)(V,E)
  • The set VVV is the vertices or nodes of GGG
  • The set E⊆V×VE \subseteq V \times VE⊆V×V is the edges of GGG
  • For every (u,v)∈E(u,v) \in E(u,v)∈E we have (v,u)∈E(v,u) \in E(v,u)∈E - symmetric
  • There are no edges (v,v)(v,v)(v,v) - irreflexive

This type of graph is called loop-free undirected graph, referring to irreflexive and symmetric.

Graph Visualization​

We usually visualize graphs as a set of points (vertices) connected by line (edges).

Neighbors​

def N(G, u):
    """
    Get the neighbors of a vertex.
    G: a graph (V, E)
    u: a vertex
    """
    V, E = G
    return {v for v in V if (u, v) in E}

def NS(G, S):
    """
    Get the neighbors of a set of vertices.
    G: a graph (V, E)
    S: a set of vertices
    """
    neighbors = set()
    for u in S:
        neighbors = neighbors.union(N(G, u))
    return neighbors

def NS_2(G, S):
    V, E = G
    return {v for v in V for u in S if (u,v) in E}

Graph Examples​

  • The set of computers in a network, where uuu and vvv are adjacent if they are connected physically (Ethernet, wifi).
  • The set of cities in the world, uuu and vvv are adjacent if there is a direct flight.

Handshaking Lemma​

Since the sum of degrees is even, there must be an even number of vertices with odd degree.
