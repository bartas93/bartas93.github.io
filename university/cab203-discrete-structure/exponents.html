Exponents

Definition: The exponent of a number says how many times to use the number in a multiplication.

  • Notation: a3a^3a3 means a×a×aa \times a \times aa×a×a.
  • ana^nan means multiply aaa by nnn times.
  • aaa is called the base.
  • nnn is called the exponent.
note

Negative exponent example:

5−35^{-3}5−3 could be calculated like:

153=15×5×5=1125\frac{1}{5^3} = \frac{1}{5 \times 5 \times 5}=\frac{1}{125}531​=5×5×51​=1251​
info

What if the Exponent is 0?

If the exponent is 0, then the answer is 1. (example 50=15^0 = 150=1)

Laws of Exponents​

  • (ab)n=an×bn(ab)^n = a^n \times b^n(ab)n=an×bn
  • am×an=am+na^m \times a^n = a^{m + n}am×an=am+n
  • am−n=amana^{m - n} = \frac{a^m}{a^n}am−n=anam​ (when a≠0a \not= 0a=0)
  • a−n=1ana^{-n} = \frac{1}{a^n}a−n=an1​ (when a≠0a \not= 0a=0)
  • a0=1a^0 = 1a0=1
  • (am)n=am×n(a^m)^n = a^{m \times n}(am)n=am×n

Exponent Examples​

  • 2102^{10}210 is the number of bytes in a kilobyte.
  • 23=82^3 = 823=8 is the number of bits in a byte.
  • 210×23=210+3=81922^{10} \times 2^{3} = 2^{10 + 3} = 8192210×23=210+3=8192 is the number of bits in a kilobyte.
