实用终端

在 eide 提供的终端中执行程序

EIDE 终端​

我们已经导出了一些 变量，程序路径 到 eide 终端上下文环境

您可以通过这个终端直接执行/访问这些 eide 内置的程序

打开一个 EIDE Terminal，使用如下 Shell 命令可以列出所有 环境变量：

  • Windows (Powershell)
  • Windows (CMD)
  • Linux
命令：ls env:
命令：set
命令：env
